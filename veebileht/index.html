HTML & CSS Fundamentals

HTML (HyperText Markup Language) provides the **structure** of a web page. Every page starts with <!DOCTYPE html> and contains a <head> (metadata) and a <body> (visible content).

CSS (Cascading Style Sheets) controls the **visual presentation** of HTML elements. We use properties like padding, border, and **Flexbox** to create responsive and adaptable layouts easily.
