What is HTML and CSS?

HTML

- HTML stands for Hyper Text Markup Language

- HTML is the standard markup language for creating Web pages

- HTML describes the structure of a Web page

- HTML consists of a series of elements

CSS

- CSS is the language we use to style a Web page.

- CSS stands for Cascading Style Sheets

- CSS saves a lot of work. It can control the layout of multiple web pages all at onceExternal stylesheets are stored in CSS files

Commands

  • ls - shows the contents of the current folder
  • cd "folder name" - changes directory (moves into the folder)
  • mkdir "folder name" - makes a new directory (creates a new folder)
  • cd Documents - Moves into the Documents folder.
  • git clone git@github.com.... - Downloads an existing project from GitHub to your local machine.
  • git add . - Stages all changes in the current directory for the next commit.
  • git commit -m "message describing changes" - Records the staged changes to the repository with a descriptive message.
  • git push - Uploads the committed changes from your local repository to the remote repository (GitHub).
  • git pull - Fetches and merges changes from the remote repository (GitHub) into your local repository.
